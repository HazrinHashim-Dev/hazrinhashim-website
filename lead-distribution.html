📖 Panduan Setup

1. Upload Google Sheet

Upload file LeadFlow_Database.xlsx ke Google Drive hang. Right-click → Open with Google Sheets.

2. Buka Apps Script

Dalam sheet tu, klik Extensions → Apps Script. Delete semua kod, paste kod dari apps-script-backend.gs. Save.

3. Run Setup

Pilih function runSetup, klik ▶ Run. Authorize bila diminta.

4. Deploy Web App

Deploy → New Deployment → Web App
Execute as: Me  |  Access: Anyone → Deploy → Copy URL

5. Sync Leads dari Drive

Paste URL di atas, klik Simpan. Lepas connected, klik "Sync dari Drive" — sistem akan baca semua spreadsheet dalam folder Drive hang dan standardize automatik.

6. Tambah Agent

Dalam Google Sheet, pergi tab Agents. Tambah nama, phone, dan code untuk setiap agent. Status = active.
