Tetapan Sistem
Sambungan Apps Script —

URL yang diperoleh selepas deploy Apps Script sebagai Web App.

Panduan Persediaan Sistem
1. Muat naik Google Sheet
Upload fail LeadFlow_Database.xlsx ke Google Drive → klik kanan → Open with Google Sheets.
2. Buka Apps Script
Dalam Sheet: Extensions → Apps Script → padam semua kod → tampal kod dari apps-script-backend.gs → Simpan.
3. Dayakan Drive API
Dalam editor Apps Script: Services (+) → cari Drive API → Add.
4. Jalankan Persediaan
Pilih fungsi runSetup → klik ▶ Jalankan → Benarkan akses.
5. Deploy sebagai Web App
Deploy → New Deployment → Web App
Execute as: Me | Access: Anyone → Deploy → Salin URL.
6. Tampal URL & Sync Data
Tampal URL di atas → Simpan & Sambung → Klik Sync dari Drive.
